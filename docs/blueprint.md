# **App Name**: SUEA Organizer

## Core Features:

- Role-Based Firebase Auth: Secure authentication for Admin and Staff roles using Firebase Auth with protected routing.
- Insight Dashboard: Visual dashboard showing employee counts, working-today status, and OT hour trends via charts.
- Cloud Registry Management: Full CRUD operations for employees, projects, and site locations utilizing Firestore collections.
- Auto-Calculation Engine: Automated logic to compute work hours, OT after 17:00, and attendance flags like late arrivals or early leavers.
- Timesheet Insight Tool: A generative AI tool that analyzes monthly hours to provide reasoning for productivity dips or unusual overtime patterns.
- Financial Reporting Suite: Generates weekly and monthly summaries, payroll summaries, and customer billing reports for print and export.

## Style Guidelines:

- Primary color: Professional Slate (#0F172A) providing a deep, corporate foundation.
- Accent: Vivid Blue (#2563EB) for highlights, actionable buttons, and data trends.
- Background: Clean, crisp White (#FFFFFF) to maintain a modern, breathable dashboard interface.
- Main font: 'Inter', a modern sans-serif selected for its neutral, machined clarity across complex data tables.
- Google Material Icons for consistent, universally understood symbols throughout the navigation and reporting modules.
- Fixed sidebar navigation for efficient context switching between staff management, timesheets, and settings.
- Smooth transitions and staggered fades when loading dashboard charts and analytical reports.
